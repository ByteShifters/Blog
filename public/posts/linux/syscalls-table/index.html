Overview

This table lists the Linux system call numbers for the x86-64 architecture, starting from 2013, and provides detailed information such as the syscall name, arguments, and direct links to their manual pages and implementation in the kernel source code.

Data Source

The data is extracted from the syscall_64.tbl file in the Linux 6.7 kernel source and the syscalls.h header file. The syscalls themselves are implemented as functions named in the Entry Point column and generated using the DEFINE_SYSCALLx macros. You can explore these details further by reviewing the Documentation/process/adding-syscalls.rst file in the kernel documentation.
